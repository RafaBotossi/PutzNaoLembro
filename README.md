# PutZ Não Lembro!

Site estático para anotar lugares, coisas e pessoas de uma campanha de D&D.

## Funcionalidades

- Categorias: Lugares, Coisas e Pessoas
- Registro com título, texto e imagem
- Criar, editar, inativar, reativar, remover, buscar e filtrar registros
- Filtro para ver registros inativos
- Imagens compactadas no navegador com limite de 200 KB por imagem
- Exportar e importar backup em JSON

## Como os dados são salvos

Os registros ficam no `localStorage` do navegador. Para levar os dados para outro computador ou guardar uma cópia, use **Exportar**. Para restaurar, use **Importar**.
